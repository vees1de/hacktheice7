# Fullstack Template

Моно-репозиторий включает три отдельных приложения:

- **backend/** — NestJS API и Python-сервис интеллектуального поиска льгот.
- **frontend/** — клиентский портал для граждан на Vue 3.
- **admin/** — админ/кассовый интерфейс для партнёров.

Каждый проект имеет собственный README с инструкциями по настройке окружения,
переменным `.env` и сценариям запуска. Для быстрого старта:

1. Ознакомьтесь с соответствующим документом:
   - [`backend/README.md`](backend/README.md)
   - [`frontend/README.md`](frontend/README.md)
   - [`admin/README.md`](admin/README.md)
2. Скопируйте `.env` из примеров и пропишите свои значения.
3. Установите зависимости (`npm ci`) и запустите нужный сервис (`npm run dev`).

CI настроен отдельными workflow для каждой части (см. `.github/workflows`).
